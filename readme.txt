Projet AMIO by Terry Dervaux and Victor Schwartz

Fonctionnalité:
 - Affichage dynamique des capteurs avec rafraichissement automatique
(couleur : vert = allumé, noir = éteind, gris = pas d'information)
 - Démarrage automatiques des tache de rafraichissement si l'option est selectionné
 - choix des parametre de notification suivant les plages horaires
 - envoie automatique d'email sans validation de l'utilisateur
 - personnalisation de l'email pour les alertes
